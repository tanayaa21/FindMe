# FindMe

AI-based missing person identification system using computer vision.

## Overview
FindMe is an AI-powered surveillance system that assists in identifying missing persons using CCTV footage. The system employs computer vision techniques and pre-trained Convolutional Neural Networks (CNNs) to detect and match faces against stored records.

When a match is found, the system can generate alerts to assist authorities in timely identification and response.

## Features
- Face detection and recognition using CNNs
- Matches CCTV footage with stored databases
- AI-assisted surveillance for public safety
- Designed to support law enforcement efforts

## Tech Stack
- Computer Vision  
- Convolutional Neural Networks (CNN)  
- Artificial Intelligence  
- Web Application  

## Use Case
FindMe helps enhance public safety by supporting missing person identification through automated video surveillance and AI-based analysis.
